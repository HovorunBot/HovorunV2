## 0.1.0

🚀 **Welcome to Hovorun v0.1.0!** Here are the core features ready to use in this chat:

* 📹 **Rich Media:** Automatically extracts and displays media from Twitter, Instagram, TikTok, Threads, Bluesky, and
  Facebook links.
* 🌍 **Auto-Translation:** Smart language detection and translation to keep chats flowing.
* 🔔 **Update Alerts:** You'll automatically get a heads-up here whenever new features drop.
* 🔑 **Group Access**: Want to use the bot in your own group? Just send allow_chat to request permission and join the
  queue.

*(P.S. If you've already been using the bot, sorry for stating the obvious! This message is just a side effect of
turning on our new automatic update notifications.)*
